# 🚗 Car Detection using YOLOv8

A web application that detects cars in uploaded images using a custom-trained YOLOv8 model. The model identifies cars, draws bounding boxes around them, and displays the total number of detected vehicles through a simple Streamlit interface.

## Features

- Car detection using a custom-trained YOLOv8 model
- Bounding box visualization
- Vehicle count
- Adjustable confidence threshold
- Clean Streamlit interface

## Built With

- Python
- YOLOv8 (Ultralytics)
- Streamlit
- OpenCV
- Pillow

## Model

The model is trained exclusively for **car detection**. It processes uploaded images and highlights each detected car with a bounding box while reporting the total number of detections.
